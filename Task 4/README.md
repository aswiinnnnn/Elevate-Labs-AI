# Classification: Logistic Regression (Breast Cancer)

## Objective
Train and evaluate a Logistic Regression classifier to detect malignant vs benign tumors using the provided dataset.

## Tools & Libraries
- Python
- Pandas
- NumPy
- Scikit-learn (train_test_split, StandardScaler, LogisticRegression)
- Matplotlib
- Seaborn

## Dataset
- **File**: `data.csv` (included in `Task 4` directory)
- The notebook drops non-feature columns (`id`, `Unnamed: 32`) and maps the diagnosis labels `M` -> 1 (malignant), `B` -> 0 (benign).

## Analysis Completed (what's implemented in `AIML_T4.ipynb`)

1. Data loading & cleaning
   - Loaded `data.csv` with Pandas.
   - Dropped `id` and `Unnamed: 32` columns.
   - Encoded `diagnosis` to numeric labels (`M`=1, `B`=0).

2. Feature / target split
   - Separated features `X` and target `y` (`diagnosis`).

3. Train / test split
   - Split data into training and test sets using an 80/20 split (`train_test_split(..., test_size=0.2, random_state=42)`).

4. Feature scaling
   - Standardized features with `StandardScaler()` (fit on training set, transform on test set).

5. Model training
   - Trained `LogisticRegression(max_iter=500)` on the scaled training data.

6. Predictions & probabilities
   - Generated class predictions (`predict`) and positive-class probabilities (`predict_proba[:, 1]`) on the test set.

7. Evaluation
   - Computed `accuracy`, `precision`, and `recall` scores.
   - Printed the `classification_report` showing precision, recall, f1-score per class.
   - Created and plotted the confusion matrix using Seaborn heatmap.
